# Multiloader Docs

This directory contains the reusable migration packet for the CarryBabyAnimals multiloader refactor pilot.

- `migration-packet.md`: repeatable workflow for Tyler's other mod repos.
- `fabric-touchpoints.md`: CarryBabyAnimals-specific Fabric touchpoint inventory.
- `boundary-rules.md`: rules for common/platform ownership.
- `verification-matrix.md`: checks required before NeoForge work begins.
- `lessons-learned.md`: reusable lessons from the pilot.
